# 📋 Kanban Board

A simple and responsive **Kanban Board** built using **HTML, CSS, and Vanilla JavaScript**. The application allows users to create tasks, drag and drop them between workflow columns, delete completed tasks, and automatically saves all changes using the browser's LocalStorage.

---

## 🚀 Features

- Create new task cards
- Drag and drop tasks between columns
- Delete task cards
- Persistent storage using LocalStorage
- Responsive and modern user interface
- Automatic data restoration after page refresh

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Browser LocalStorage API
- HTML Drag and Drop API
